boxes.nickcharlton.net

Vagrant is an amazing tool for short-lived and automated virtual machines, but the boxes are often quite outdated. boxes is a tool for automating building Vagrantboxes in many different styles. This is a list of recently constructed boxes, which are rebuilt every month.

Usage

Add a box as you would usually:

$ vagrant box add {title} {url}
$ vagrant init {title}
$ vagrant up

Additionally, it supports the vagrant-box-updater plugin.

A project by Nick Charlton. Source on GitHub.
